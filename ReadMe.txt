Thank you for downloading IncEditor.

IncEditor is an editor built using WPF, MVVM and MEF.

To run the application, follow these steps:

1. Open the solution in Visual Studio and build all the projects.
2. Copy and paste the .dll binaries of DefaultTheme and ExpressionTheme into the IncEditor\Themes folder.
3. Now run, the application and you're ready to go!

IncEditor is a program that is built mainly for educational purposes and secondarily for replacing the notepad. If you want to learn WPF/MVVM/MEF, read the source code and try to learn and get involved in the project. If you know about WPF/MVVM/MEF, help make IncEditor better and help others learn!
